# DGR-FRAM Analyze

Plataforma de modelagem sociotécnica baseada no FRAM (Functional Resonance Analysis Method) integrada à Governança de Dados e Engenharia de Resiliência.

## Objetivo

Analisar variabilidades, mecanismos de amortecimento e propagação da ressonância funcional em infraestruturas críticas.

## Funcionalidades

- Cadastro de funções FRAM
- Cadastro de acoplamentos
- Avaliação de variabilidade endógena
- Avaliação de variabilidade exógena
- Avaliação de mecanismos de amortecimento
- Cálculo da Variabilidade Funcional Residual (VFR)
- Simulação Monte Carlo
- Dashboard de Resiliência
- Exportação de relatórios

## Modelo Matemático

VFRᵢ = (VEᵢ + VXᵢ) × (1 − ARᵢ)

Onde:

- VE = Variabilidade Endógena
- VX = Variabilidade Exógena
- AR = Amortecimento Resiliente
- VFR = Variabilidade Funcional Residual
